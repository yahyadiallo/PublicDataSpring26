# D3 Multi-Line Chart

## Overview
This project uses D3 to create a multi-line chart showing monthly temperature data for New York City in 2023.

The chart loads data from an external CSV file and visualizes the maximum and minimum temperatures over time using SVG paths and D3 line generators.

## Files
- `index.html` - main webpage
- `style.css` - chart styling
- `main.js` - D3 visualization code
- `nyc-2023.csv` - temperature dataset

## Visualization
The visualization includes:
- Two line charts
- Time scale on the x-axis
- Temperature scale on the y-axis
- SVG title text
- CSS styling for lines and text
