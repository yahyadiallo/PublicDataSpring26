# EA Sports FC 24 Review Website

## Overview
This project is a simple HTML and CSS webpage that displays a public-facing review of the game *EA Sports FC 24*. The goal was to practice basic web development skills like structuring content with HTML and styling it with CSS.

---

## What the HTML does
The `index.html` file is the structure of the webpage.

- It includes a title, heading, and author name
- It displays an image of the game cover
- It contains paragraphs with the review text
- It includes a link to the official EA Sports FC 24 website
- Comments are included to explain sections of the code

---

## What the CSS does
The `style.css` file controls how the page looks.

- Changes the font using Google Fonts (Source Sans 3)
- Centers the content using the `.review` class
- Styles the title using an `id` selector (`#title`)
- Adjusts font size, color, and spacing for readability
- Centers and styles the image
- Styles links and adds a hover effect
